🎬 Online Movie Ticket Booking System
A web-based movie ticket booking system that allows users to view available movies, book tickets, and make payments online.
This project is built using PHP, MySQL, HTML, and CSS.
________________________________________
📌 Features
✔ User Registration and Login
✔ View Available Movies
✔ Book Movie Tickets
✔ Ticket Quantity Selection
✔ Payment Processing
✔ Booking Confirmation
________________________________________

⚙️ System Workflow
1️⃣ User registers or logs into the system
2️⃣ System displays available movies
3️⃣ User selects a movie
4️⃣ User enters number of tickets
5️⃣ System calculates total price
6️⃣ User makes payment
7️⃣ Booking confirmation is displayed
